Установка:

1 вариант
Разархивируйте содержимое архива в папку Mods в корневой папке игры

Пример верного пути после установки: Mods\Kotyanka\Cat_Rangers\DATA\Script\Cat_ZelRangers.scr

Если по указанному пути (без лишних папок) оказался указанный файл, то установка была выполнена верно!

2 вариант
Запустите установщик (папка "Для тех, у кого лапки") и проследуйте инструкциям установщика.
Установщик выполнит всё автоматически

Если у вас нет ни Steam, ни GOG-версии игры, то укажите установщику корневую папку игры
Дальше установщик сделает всё сам

Installation:

1 way
Dearchive Kotyanka.7z into Mods folder in game's main folder

2 way
Double-click on Kotyanka.exe (folder "Для тех, у кого лапки") and follow installer's instructions
Installer will make everything by itself

If you don't have Steam or GOG, set way to game's folder, after that installer'll make everything by himself
