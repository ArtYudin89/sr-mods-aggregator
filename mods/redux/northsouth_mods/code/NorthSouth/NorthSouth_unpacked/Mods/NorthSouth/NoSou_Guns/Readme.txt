Во всём файле разложены текстовики с подробными объяснениями

Чтобы мод заработал, требуется, используя BlockParEditor, сохранить все .txt файлы в папке CFG и подпапке Rus сохранить как файлы .dat
Кроме того, используя актуальный RScript (использовался RScript 2.12 modified by marshalska), нужно будет скомпилировать скрипт (бишь файл .rson)

		ПРЕДВАРИТЕЛЬНО
Уберите из файлов все пустые строчки (где нет совершенно никакого текста, даже одинокой скобочки ( } )), а также полностью удалите
все строки с комментариями (Это строчки, содержащие в себе //)

Если вы не хотите добавлять уникальную графику для микромодуля, то удалите папку DATA целиком, а также файл CacheData.txt и CacheData.dat
в папке CFG, а также Main.txt и Main.dat
