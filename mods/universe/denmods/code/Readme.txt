Список модов:
    DenSettingsControl
    - Добавляет панель, с помощью которой можно менять настройки партии

    DenButtonsPack
    - Добавляет графику различных кнопок, которые могут использоваться в других модах

    DenDynamicBG
    - Разные фоны на планетах и станциях

    DenBG
    - Пак с фонам для мода на динамические фоны

    DenNightBG
    - Добавляет 8 ночных фонов планет

    DenUIRecolor
    - Перекрашивает интерфейс игры в один из цветов

    DenDatVersion
    - Добавляет возможность запускать игру версии любой поддерживаемой версии с любыми датниками

    DenGalaxyMapKlingColors
    - Показывает серию доминаторов на галакарте
